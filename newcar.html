עזרים נלווים בסיסיים

  1. בתא הנוסעים
    1. אפוד זוהר
    2. פח זבל
    3. שטיחי גומי מתחת למושבים (2 קידמיים + 2 אחוריים)
    4. משענות ראש (2 קידמיות + 3 אחוריות)
    5. מראה אחורית

  2. בתא המטען
    1. גלגל רזרבי
    2. ג'ק + ידית
    3. מפתח לגלגלים
    4. מתאם מתברג לגרירה
    5. משולש עצירה

  3. אביזרים חיצוניים
    1. ווישרים (קדמיים ואחורי)
    2. מראות צד (ימנית ושמאלית + מכסים תואמים לצבע הרכב)

  4. אמצעי מיגון
    1. סט מפתחות לרכב (2 עם שלט + מפתח מוסכים)
    2. נעילה מרכזית (שלט + מפתח)
    3. אמוביליזר מובנה במפתח הרכב
    4. קודן + קוד + הוראות החלפת קוד
    5. כספת למחשב הרכב (אמצעי נגד גניבה)

תוספות

  1. מנגנונים
    1. מרימי חלונות בסגירת הרכב, בהפעלת נעילה מרכזית (שלט + מפתח)
    2. גלאי רברס + מד מרחק
    3. מצלמת רברס + פסי מרחק ומדרכה על המסך
    4. מצלמת תיעוד הנסיעה
    5. כיבוי אוטומתי של אורות בנעילת הרכב
    6. גלאי אור (להדלקה אוטומתית של אורות בחשיכה)
    7. גלאי גשם (להפעלה אוטומתית של ווישרים בגשם)

  2. גלגלים
    1. גלגלים רגילים
      1. טסות + סט רצועות מקשרות לאבטחת טסות
    2. גלגלי מגנזיום
      1. סט ברגים מיוחדים לגלגלים נגד גניבה + מתאמים למפתח

  3. מגינים
    1. מגן שמשה קידמית ו/או אחורית (נגד השמש)
    2. מגיני צד לדלתות צדדיות
    3. מגיני פגוש קידמי ואחורי
    4. סט מגיני מדבקות שקופות (בפגוש אחורי להוצאת מטען, מתחת לדלתות צד, אחרים)

  4. אחר
    1. כיסויי מושבים
    2. מגש גומי או פלסטיק בתא המטען
    3. סט מנורות חליפיות
    4. כבלים להצטה מרכב אחר

הדרכה

  1. פתיחה וסגירה של הרכב

    2. חלונות
    3. מראות צד
    4. מכסה מנוע

  2. הפעלת מנועה
    1. מצב תיבת הילוכים
    2. מצב דוושת ברקס
    3. מצב דוושת קלצ'
    4. מצב מפתח הרכב במנעול הסטרטר

  3. מצבי נסיעה ותיבת הילוכים
    1. נסיעה קדימה
      1. מצב רגיל
      2. מניעת מעבר להילוך גבוהה (בירידה)
      3. מצב ספורט (העצה / הקפה)
      4. הדרדרות בהתנעה ממצב עצירה בעלייה תלולה (זינוק בעלייה)
    2. נסיעה אחורה
      1. הדגמת מצלמה אחורית
      2. הדגמת גלאיי ריברס
    3. מצב חנייה

  4. הדלקת אורות

    4. פנסי ערפל
      1. קידמי
      2. אחורי
    5. הפעלה / כיבוי של מנגנון להפעלה אוטומתית של אורות בחשיכה
    6. הפעלה / כיבוי של מנגנון לכיבוי אורות בנעילת הרכב
    7. בלינקרים
    8. אורות עצירת חירום

  5. הפעלת ווישרים
    1. ווישרים קידמיים
      1. שטיפה
      2. הפעלה עם קצב משתנה
      3. הפעלה בקצב קבוע נמוך
      4. הפעלה בקצב קבוע גבוהה
      5. הפעלה / כיבוי של מנגנון להפעלה אוטומתית של ווישרים בגשם
    2. ווישר אחורי
      1. שטיפה
      2. הפעלה לסירוגים
      3. הפעלה בקצב קבוע
      4. הפעלה / כיבוי של מנגנון להפעלה אוטומתית של ווישר בגשם

  6. כווונון מראות

    2. מראות צדדיות

  7. כריות אוויר
    1. מיקומם
    2. ניגודים עם כיסויי מושבים (לכריות צדדיות)
    3. ביטול / הפעלה של כרית במושב קידמי של נוסע (להסעת סלקל)

  8. הפעלת מערכת אוורור ו/או מזגן

    2. הפשרה מהירה של שמשה האחורית
    3. הפעלה / כיבוי מזגן
    4. כיוונון של אוויר למעלה / למטה / לחלונות
    5. כיוונון של אוויר לנהג ולנוסע הקידמי
    6. כיוונון של אוויר לחלק אחורי של תא נוסעים

    5. דיסק

    6. USB

    7. Bluetooth

  1. מדבקות שונות (חנייה, וכו')
  2. מראה אחורית פנורמית
  3. מעמד לטלפון ניצמד לשמשה קדמית
  4. מצלמת תיעוד הנסיעה
  5. חפצים מתא הכפפות
  6. חפצים מתא ליד תיבת הילוכים
  7. חפצים מתאים בדלתות (קידמיות ואחוריות)
  8. חפצים מכיסים בגב משענות של כסאות קדמיים
  9. חפצים מתא המטען
